# Decision Assurance Layer for AI-Assisted Cybersecurity

IBM AI Builders Challenge July Wildcard project.

## Project Thesis

Future cybersecurity is not AI autonomy. Future cybersecurity is accountable AI-assisted command and control.

## Problem

AI is becoming embedded in cybersecurity workflows, but organizations need a way to preserve evidence, human judgment, uncertainty tracking, and auditability.

## Proposed Solution

This project proposes a Decision Assurance Layer for AI-assisted cybersecurity operations.
